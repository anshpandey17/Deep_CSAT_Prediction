# 📊 Customer Satisfaction (CSAT) Prediction using Machine Learning & Deep Learning

## 🚀 Project Overview

Customer satisfaction is a key indicator of service quality and business performance.
This project builds a **Machine Learning and Deep Learning pipeline** to predict **Customer Satisfaction (CSAT) scores** using customer support interaction data.

The project involves **data preprocessing, feature engineering, NLP-based sentiment analysis, model training, and deployment using Streamlit**.

---

## 🎯 Problem Statement

Businesses receive thousands of customer support interactions daily. Understanding which factors impact **customer satisfaction** helps organizations improve service quality.

The goal of this project is to **predict CSAT scores based on support interaction features such as response time, survey delay, and sentiment scores**.

---

## 📂 Dataset Features

Key features used for prediction include:

* ⏱️ **Response Time**
* 🕒 **Survey Delay**
* 💬 **Customer Sentiment Score**
* 📊 **Customer Support Interaction Data**
